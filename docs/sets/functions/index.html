Functions

(Content adapted from Critchlow & Eck)

Both the real world and the world of mathematics are full of what are called, in mathematics, "functional relationships." A functional relationship is a relationship between two sets, which associates exactly one element from the second set to each element of the first set.

For example, each item for sale in a store has a price. The first set in this relationship is the set of items in the store. For each item in the store, there is an associated price, so the second set in the relationship is the set of possible prices. The relationship is a functional relationship because each item has a price. That is, the question "What is the price of this item?" has a single, definite answer for each item in the store.

Similarly, the question "Who is the (biological) mother of this person?" has a single, definite answer for each person. So, the relationship "mother of" defines a functional relationship. In this case, the two sets in the relationship are the same set, namely the set of people.1 On the other hand, the relationship "child of" is not a functional relationship. The question "Who is the child of this person?" does not have a single, definite answer for each person. A given person might not have any child at all. And a given person might have more than one child. Either of these cases—a person with no child or a person with more than one child—is enough to show that the relationship "child of" is not a functional relationship.

Or consider an ordinary map, such as a map of New York State or a street map of Rome. The whole point of the map, if it is accurate, is that there is a functional relationship between points on the map and points on the surface of the Earth. Perhaps because of this example, a functional relationship is sometimes called a mapping.

There are also many natural examples of functional relationships in mathematics. For example, every rectangle has an associated area. This fact expresses a functional relationship between the set of rectangles and the set of numbers. Every natural number nnn has a square, n2n^2n2. The relationship "square of" is a functional relationship from the set of natural numbers to itself.

In mathematics, of course, we need to work with functional relationships in the abstract. To do this, we introduce the idea of function. You should think of a function as a mathematical object that expresses a functional relationship between two sets. The notation f ⁣:A→Bf\colon A\to Bf:A→B expresses the fact that fff is a function from the set AAA to the set BBB. That is, fff is a name for a mathematical object that expresses a functional relationship from the set AAA to the set BBB. The notation f ⁣:A→Bf\colon A\to Bf:A→B is read as "fff is a function from AAA to BBB" or more simply as "fff maps AAA to BBB."

If f ⁣:A→Bf\colon A\to Bf:A→B and if a∈Aa\in Aa∈A, the fact that fff is a functional relationship from AAA to BBB means that fff associates some element of BBB to aaa. That element is denoted f(a)f(a)f(a). That is, for each a∈Aa\in Aa∈A, f(a)∈Bf(a)\in Bf(a)∈B and f(a)f(a)f(a) is the single, definite answer to the question "What element of BBB is associated to aaa by the function f f\,f?" The fact that fff is a function from AAA to BBB means that this question has a single, well-defined answer. Given a∈Aa\in Aa∈A, f(a)f(a)f(a) is called the value of the function fff at aaa.

For example, if III is the set of items for sale in a given store and MMM is the set of possible prices, then there is function c ⁣:I→Mc\colon I\to Mc:I→M which is defined by the fact that for each x∈Ix\in Ix∈I, c(x)c(x)c(x) is the price of the item xxx. Similarly, if PPP is the set of people, then there is a function m ⁣:P→Pm\colon P\to Pm:P→P such that for each person ppp, m(p)m(p)m(p) is the mother of ppp. And if N\NN is the set of natural numbers, then the formula s(n)=n2s(n) = n^2s(n)=n2 specifies a function s ⁣:N→Ns\colon \N\to\Ns:N→N. It is in the form of formulas such as s(n)=n2s(n)=n^2s(n)=n2 or f(x)=x3−3x+7f(x)=x^3-3x+7f(x)=x3−3x+7 that most people first encounter functions. But you should note that a formula by itself is not a function, although it might well specify a function between two given sets of numbers. Functions are much more general than formulas, and they apply to all kinds of sets, not just to sets of numbers.

#Composition

#Pairs

If aaa and bbb are entities, then (a,b)(a,b)(a,b) denotes the ordered pair containing aaa and bbb. The ordered pair (a,b)(a,b)(a,b) differs from the set {a,b}\{a,b\}{a,b} because a set is not ordered. That is, {a,b}\{a,b\}{a,b} and {b,a}\{b,a\}{b,a} denote the same set, but if a≠ba\not=ba​=b, then (a,b)(a,b)(a,b) and (b,a)(b,a)(b,a) are different ordered pairs. More generally, two ordered pairs (a,b)(a,b)(a,b) and (c,d)(c,d)(c,d) are equal if and only if both a=ca=ca=c and b=db=db=d. If (a,b)(a,b)(a,b) is an ordered pair, then aaa and bbb are referred to as the coordinates of the ordered pair. In particular, aaa is the first coordinate and bbb is the second coordinate.

If AAA and BBB are sets, then we can form the set A×BA\times BA×B which is defined by

A×B={(a,b)∣a∈A and b∈B}.A\times B= \{(a,b)\mid a\in A \text{ and } b\in B\}.A×B={(a,b)∣a∈A and b∈B}.

This set is called the cross product or Cartesian product of the sets AAA and BBB. The set A×BA\times BA×B contains every ordered pair whose first coordinate is an element of AAA and whose second coordinate is an element of BBB. For example, if X={c,d}X=\{c,d\}X={c,d} and Y={1,2,3}Y=\{1,2,3\}Y={1,2,3}, then X×Y={(c,1),(c,2),(c,3),(d,1),(d,2),(d,3)}X\times Y=\{(c,1), (c,2), (c,3), (d,1),(d,2), (d,3)\}X×Y={(c,1),(c,2),(c,3),(d,1),(d,2),(d,3)}. It is possible to extend this idea to the cross product of more than two sets. The cross product of the three sets AAA, BBB, and CCC is denoted A×B×CA\times B\times CA×B×C. It consists of all ordered triples (a,b,c)(a,b,c)(a,b,c) where a∈Aa\in Aa∈A, b∈Bb\in Bb∈B, and c∈Cc\in Cc∈C. The definition for four or more sets is similar. The general term for a member of a cross product is tuple or, more specifically, ordered n-tuple. For example, (a,b,c,d,e)(a,b,c,d,e)(a,b,c,d,e) is an ordered 5-tuple.

Given a function f ⁣:A→Bf\colon A\to Bf:A→B, consider the set {(a,b)∈A×B∣a∈A and b=f(a)}\{(a,b)\in A\times B\mid a\in A \text{ and } b=f(a)\}{(a,b)∈A×B∣a∈A and b=f(a)}. This set of ordered pairs consists of all pairs (a,b)(a,b)(a,b) such that a∈Aa\in Aa∈A and bbb is the element of BBB that is associated to aaa by the function fff. The set {(a,b)∈A×B∣a∈A and b=f(a)}\{(a,b)\in A\times B\mid a\in A \text{ and } b=f(a)\}{(a,b)∈A×B∣a∈A and b=f(a)} is called the graph of the function fff. Since fff is a function, each element a∈Aa\in Aa∈A occurs once and only once as a first coordinate among the ordered pairs in the graph of fff. Given a∈Aa\in Aa∈A, we can determine f(a)f(a)f(a) by finding that ordered pair and looking at the second coordinate. In fact, it is convenient to consider the function and its graph to be the same thing, and to use this as our official mathematical definition.2

Let AAA and BBB be sets. A function from AAA to BBB is a subset of A×BA\times BA×B which has the property that for each a∈Aa\in Aa∈A, the set contains one and only one ordered pair whose first coordinate is aaa. If (a,b)(a,b)(a,b) is that ordered pair, then bbb is called the value of the function at aaa and is denoted f(a)f(a)f(a). If b=f(a)b=f(a)b=f(a), then we also say that the function fff maps aaa to bbb. The fact that fff is a function from AAA to BBB is indicated by the notation f ⁣:A→Bf\colon A\to Bf:A→B.

For example, if X={a,b}X=\{a,b\}X={a,b} and Y={1,2,3}Y=\{1,2,3\}Y={1,2,3}, then the set {(a,2),(b,1)}\{(a,2), (b,1)\}{(a,2),(b,1)} is a function from XXX to YYY, and {(1,a),(2,a),(3,b)}\{(1,a), (2,a), (3,b)\}{(1,a),(2,a),(3,b)} is a function from YYY to XXX. On the other hand, {(1,a),(2,b)}\{(1,a),(2,b)\}{(1,a),(2,b)} is not a function from YYY to XXX, since it does not specify any value for 3. And {(a,1),(a,2),(b,3)}\{(a,1),(a,2),(b,3)\}{(a,1),(a,2),(b,3)} is not a function from XXX to YYY because it specifies two different values, 1 and 2, associated with the same element, aaa, of XXX.

Even though the technical definition of a function is a set of ordered pairs, it's usually better to think of a function from AAA to BBB as something that associates some element of BBB to every element of AAA. The set of ordered pairs is one way of expressing this association. If the association is expressed in some other way, it's easy to write down the set of ordered pairs. For example, the function s ⁣:N→Ns\colon\N\to\Ns:N→N which is specified by the formula s(n)=n2s(n)=n^2s(n)=n2 can be written as the set of ordered pairs {(n,n2)∣n∈N}\{(n,n^2)\mid n\in \N\}{(n,n2)∣n∈N}.

#Additional Concepts

Suppose that f ⁣:A→Bf\colon A\to Bf:A→B is a function from the set AAA to the set BBB. We say that AAA is the domain of the function fff and that BBB is the range of the function. We define the image of the function fff to be the set {b∈B∣∃a∈A (b=f(a))}\{b\in B\mid \exists a\in A\,(b=f(a))\}{b∈B∣∃a∈A(b=f(a))}. Put more simply, the image of fff is the set {f(a)∣a∈A}\{f(a)\mid a\in A\}{f(a)∣a∈A}. That is, the image is the set of all values, f(a)f(a)f(a), of the function, for all a∈Aa\in Aa∈A. (You should note that in some cases—particularly in calculus courses—the term "range" is used to refer to what I am calling the image.) For example, for the function s ⁣:N→Ns\colon\N\to\Ns:N→N that is specified by s(n)=n2s(n)=n^2s(n)=n2, both the domain and the range are N\NN, and the image is the set {n2∣n∈N}\{n^2\mid n\in\N\}{n2∣n∈N}, or {0,1,4,9,16,… }\{0,1,4,9,16,\dots\}{0,1,4,9,16,…}.

Note that the image of a function is a subset of its range. It can be a proper subset, as in the above example, but it is also possible for the image of a function to be equal to the range. In that case, the function is said to be onto. Sometimes, the fancier term surjective is used instead. Formally, a function f ⁣:A→Bf\colon A\to Bf:A→B is said to be onto (or surjective) if every element of BBB is equal to f(a)f(a)f(a) for some element of AAA. In terms of logic, fff is onto if and only if

∀b∈B (∃a∈A (b=f(a))).\forall b\in B\,\big(\exists a\in A\, (b=f(a))\big).∀b∈B(∃a∈A(b=f(a))).

For example, let X={a,b}X=\{a,b\}X={a,b} and Y={1,2,3}Y=\{1,2,3\}Y={1,2,3}, and consider the function from YYY to XXX specified by the set of ordered pairs {(1,a),(2,a),(3,b)}\{(1,a),(2,a),(3,b)\}{(1,a),(2,a),(3,b)}. This function is onto because its image, {a,b}\{a,b\}{a,b}, is equal to the range, XXX. However, the function from XXX to YYY given by {(a,1),(b,3)}\{(a,1),(b,3)\}{(a,1),(b,3)} is not onto, because its image, {1,3}\{1,3\}{1,3}, is a proper subset of its range, YYY. As a further example, consider the function fff from Z\ZZ to Z\ZZ given by f(n)=n−52f(n) = n-52f(n)=n−52. To show that fff is onto, we need to pick an arbitrary bbb in the range Z\ZZ and show that there is some number aaa in the domain Z\ZZ such that f(a)=bf(a) = bf(a)=b. So let bbb be an arbitrary integer; we want to find an aaa such that a−52=ba-52=ba−52=b. Clearly this equation will be true when a=b+52a=b+52a=b+52. Note that if fff had been specified to have domain N\NN, then fff would not be onto, as for some b∈Zb \in \Zb∈Z the number a=b+52a=b+52a=b+52 is not in the domain N\NN (for example, the integer −73-73−73 is not in the image of fff, since −21-21−21 is not in N\NN.)

If f ⁣:A→Bf\colon A\to Bf:A→B and if a∈Aa\in Aa∈A, then aaa is associated to only one element of BBB. This is part of the definition of a function. However, no such restriction holds for elements of BBB. If b∈Bb\in Bb∈B, it is possible for bbb to be associated to zero, one, two, three, \dots, or even to an infinite number of elements of AAA. In the case where each element of the range is associated to at most one element of the domain, the function is said to be one-to-one. Sometimes, the term injective is used instead. The function fff is one-to-one (or injective) if for any two distinct elements xxx and yyy in the domain of fff, f(x)f(x)f(x) and f(y)f(y)f(y) are also distinct. In terms of logic, f ⁣:A→Bf\colon A\to Bf:A→B is one-to-one if and only if

∀x∈A  ∀y∈A (x≠y→f(x)≠f(y)).\forall x\in A\,\,\forall y\in A\,\big(x\not=y\rightarrow f(x)\not=f(y)\big).∀x∈A∀y∈A(x​=y→f(x)​=f(y)).

Since a proposition is equivalent to its contrapositive, we can write this condition equivalently as

∀x∈A  ∀y∈A (f(x)=f(y)→x=y).\forall x\in A\,\,\forall y\in A\,\big(f(x)=f(y)\rightarrow x=y\big).∀x∈A∀y∈A(f(x)=f(y)→x=y).

Sometimes, it is easier to work with the definition of one-to-one when it is expressed in this form. The function that associates every person to his or her mother is not one-to-one because it is possible for two different people to have the same mother. The function s ⁣:N→Ns\colon\N\to\Ns:N→N specified by s(n)=n2s(n)=n^2s(n)=n2 is one-to-one. However, we can define a function r ⁣:Z→Zr\colon\Z\to\Zr:Z→Z by the same formula: r(n)=n2r(n)=n^2r(n)=n2, for n∈Zn\in\Zn∈Z. A function that is both one-to-one and onto is said to be bijective. The function that associates each point in a map of New York State to a point in the state itself is presumably bijective. For each point on the map, there is a corresponding point in the state, and vice versa. If we specify the function fff from the set {1,2,3}\{1,2,3\}{1,2,3} to the set {a,b,c}\{a,b,c\}{a,b,c} as the set of ordered pairs {(1,b),(2,a),(3,c)}\{(1,b),(2,a),(3,c)\}{(1,b),(2,a),(3,c)}, then fff is a bijective function. Or consider the function from Z\ZZ to Z\ZZ given by f(n)=n−52f(n) = n-52f(n)=n−52. We have already shown that fff is onto. We can show that it is also one-to-one: pick an arbitrary xxx and yyy in Z\ZZ and assume that f(x)=f(y)f(x) = f(y)f(x)=f(y). This means that x−52=y−52x-52 = y-52x−52=y−52, and adding 52 to both sides of the equation gives x=yx=yx=y. Since xxx and yyy were arbitrary, we have proved ∀x∈Z  ∀y∈Z (f(x)=f(y)→x=y)\forall x\in \Z\,\,\forall y\in \Z\,(f(x)=f(y)\rightarrow x=y)∀x∈Z∀y∈Z(f(x)=f(y)→x=y), that is, that fff is one-to-one. Altogether, then, fff is a bijection.

#First-Class Functions

One difficulty that people sometimes have with mathematics is its generality. A set is a collection of entities, but an "entity" can be anything at all, including other sets. Once we have defined ordered pairs, we can use ordered pairs as elements of sets. We could also make ordered pairs of sets. Now that we have defined functions, every function is itself an entity. This means that we can have sets that contain functions. We can even have a function whose domain and range are sets of functions. Similarly, the domain or range of a function might be a set of sets, or a set of ordered pairs. Computer scientists have a good name for this. They would say that sets, ordered pairs, and functions are first-class objects. Once a set, ordered pair, or function has been defined, it can be used just like any other entity. If they were not first-class objects, there could be restrictions on the way they can be used. For example, it might not be possible to use functions as members of sets. (This would make them "second class.")

For example, suppose that AAA, BBB, and CCC are sets. Then since A×BA\times BA×B is a set, we might have a function f ⁣:A×B→Cf\colon A\times B\to Cf:A×B→C. If (a,b)∈A×B(a,b)\in A\times B(a,b)∈A×B, then the value of fff at (a,b)(a,b)(a,b) would be denoted f((a,b))f((a,b))f((a,b)). In practice, though, one set of parentheses is usually dropped, and the value of fff at (a,b)(a,b)(a,b) is denoted f(a,b)f(a,b)f(a,b). As a particular example, we might define a function p ⁣:N×N→Np\colon \N\times\N\to\Np:N×N→N with the formula p(n,m)=nm+1p(n,m)=nm+1p(n,m)=nm+1. Similarly, we might define a function q ⁣:N×N×N→N×Nq\colon \N\times\N\times\N\to\N\times\Nq:N×N×N→N×N by q(n,m,k)=(nm−k,nk−n)q(n,m,k)=(nm-k,nk-n)q(n,m,k)=(nm−k,nk−n).

Suppose that AAA and BBB are sets. There are, in general, many functions that map AAA to BBB. We can gather all those functions into a set. This set, whose elements are all the functions from AAA to BBB, is denoted BAB^ABA. (We'll see later why this notation is reasonable.) Using this notation, saying f ⁣:A→Bf\colon A\to Bf:A→B is exactly the same as saying f∈BAf\in B^Af∈BA. Both of these notations assert that fff is a function from AAA to BBB. Of course, we can also form an unlimited number of other sets, such as the power set P(BA){\mathscr P}\big(B^A\big)P(BA), the cross product BA×AB^A\times ABA×A, or the set AA×AA^{A\times A}AA×A, which contains all the functions from the set A×AA\times AA×A to the set AAA. And of course, any of these sets can be the domain or range of a function. An example of this is the function E ⁣:BA×A→B{\mathscr E}\colon B^A\times A\to BE:BA×A→B defined by the formula E(f,a)=f(a){\mathscr E}(f,a) = f(a)E(f,a)=f(a). Let's see if we can make sense of this notation. Since the domain of E{\mathscr E}E is BA×AB^A\times ABA×A, an element in the domain is an ordered pair in which the first coordinate is a function from AAA to BBB and the second coordinate is an element of AAA. Thus, E(f,a){\mathscr E}(f,a)E(f,a) is defined for a function f ⁣:A→Bf\colon A\to Bf:A→B and an element a∈Aa\in Aa∈A. Given such an fff and aaa, the notation f(a)f(a)f(a) specifies an element of BBB, so the definition of E(f,a){\mathscr E}(f,a)E(f,a) as f(a)f(a)f(a) makes sense. The function E{\mathscr E}E is called the "evaluation function" since it captures the idea of evaluating a function at an element of its domain.
