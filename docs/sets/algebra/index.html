Boolean Algebra of Sets

(Content adapted from Critchlow & Eck)

It is clear that set theory is closely related to logic. The intersection and union of sets can be defined in terms of the logical "and" and logical "or" operators. The notation {x∣P(x)}\{x\mid P(x)\}{x∣P(x)} makes it possible to use predicates to specify sets. And if AAA is any set, then the formula x∈Ax\in Ax∈A defines a one place predicate that is true for an entity xxx if and only if xxx is a member of AAA. So it should not be a surprise that many of the rules of logic have analogs in set theory.

For example, we have already noted that ∪\cup∪ and ∩\cap∩ are commutative operations. This fact can be verified using the rules of logic. Let AAA and BBB be sets. According to the definition of equality of sets, we can show that A∪B=B∪AA\cup B=B\cup AA∪B=B∪A by showing that ∀x ((x∈A∪B)↔(x∈B∪A))\forall x\,\big((x\in A\cup B)\leftrightarrow(x\in B\cup A)\big)∀x((x∈A∪B)↔(x∈B∪A)). But for any xxx,

x∈A∪Bx\in A\cup Bx∈A∪B↔x∈A∨x∈B\leftrightarrow x\in A \lor x\in B↔x∈A∨x∈Bdefinition of ∪\cup∪
↔x∈B∨x∈A\leftrightarrow x\in B \lor x\in A↔x∈B∨x∈Acommutativity of ∨\lor∨
↔x∈B∪A\leftrightarrow x\in B \cup A↔x∈B∪Adefinition of ∪\cup∪

The commutativity of ∩\cap∩ follows in the same way from the definition of ∩\cap∩ in terms of ∧\land∧ and the commutativity of ∧\land∧, and a similar argument shows that union and intersection are associative operations.

The distributive laws for propositional logic give rise to two similar rules in set theory. Let AAA, BBB, and CCC be any sets. Then

A∪(B∩C)=(A∪B)∩(A∪C)A\cup(B\cap C)=(A\cup B)\cap(A\cup C)A∪(B∩C)=(A∪B)∩(A∪C)
A∩(B∪C)=(A∩B)∪(A∩C)A\cap(B\cup C)=(A\cap B)\cup(A\cap C)A∩(B∪C)=(A∩B)∪(A∩C)

These rules are called the distributive laws for set theory. To verify the first of these laws, we just have to note that for any xxx,

x∈A∪(B∩C)x\in A\cup(B\cap C)x∈A∪(B∩C)↔(x∈A)∨((x∈B)∧(x∈C))\leftrightarrow (x\in A)\lor((x\in B)\land (x\in C))↔(x∈A)∨((x∈B)∧(x∈C))definition of ∪\cup∪, ∩\cap∩
↔((x∈A)∨(x∈B))∧((x∈A)∨(x∈C))\leftrightarrow ((x\in A)\lor(x\in B)) \land((x \in A)\lor (x\in C))↔((x∈A)∨(x∈B))∧((x∈A)∨(x∈C))distributivity of ∨\lor∨
↔(x∈A∪B)∧(x∈A∪C)\leftrightarrow (x\in A\cup B) \land (x \in A\cup C)↔(x∈A∪B)∧(x∈A∪C)definition of ∪\cup∪
↔x∈((A∪B)∩(A∪C))\leftrightarrow x\in ((A\cup B)\cap(A\cup C))↔x∈((A∪B)∩(A∪C))definition of ∩\cap∩

The second distributive law for sets follows in exactly the same way.

#Complement

While ∪\cup∪ is analogous to ∨\lor∨ and ∩\cap∩ is analogous to ∧\land∧, we have not yet seen any operation in set theory that is analogous to the logical "not" operator, ¬\lnot¬. Given a set AAA, it is tempting to try to define {x∣¬(x∈A)}\{x\mid \lnot(x\in A)\}{x∣¬(x∈A)}, the set that contains everything that does not belong to AAA. Unfortunately, the rules of set theory do not allow us to define such a set. The notation {x∣P(x)}\{x\mid P(x)\}{x∣P(x)} can only be used when the domain of discourse of PPP is a set, so there must be an underlying set from which the elements that are/are not in AAA are chosen, i.e. some underlying set of which AAA is a subset. We can get around this problem by restricting the discussion to subsets of some fixed set. This set will be known as the universal set. Keep in mind that the universal set is only universal for some particular discussion. It is simply some set that is large enough to contain all the sets under discussion as subsets. Given a universal set UUU and any subset AAA of UUU, we can define the set {x∈U∣¬(x∈A)}\{x\in U\mid \lnot(x\in A)\}{x∈U∣¬(x∈A)}.

Let UUU be a given universal set, and let AAA be any subset of UUU. We define the complement of AAA in UUU to be the set A‾\overline{A}A that is defined by A‾={x∈U∣x∉A}\overline{A}=\{x\in U\mid x\not\in A\}A={x∈U∣x​∈A}.

Usually, we will refer to the complement of AAA in UUU simply as the complement of AAA, but you should remember that whenever complements of sets are used, there must be some universal set in the background.

Given the complement operation on sets, we can look for analogs to the rules of logic that involve negation. For example, we know that p∧¬p=Fp\land\lnot p=\Fp∧¬p=F for any proposition ppp. It follows that for any subset AAA of UUU,

A∩A‾A\cap\overline{A}A∩A={x∈U∣(x∈A)∧(x∈A‾)}= \{x\in U\mid (x\in A)\land (x\in \overline{A})\}={x∈U∣(x∈A)∧(x∈A)}definition of ∩\cap∩
={x∈U∣(x∈A)∧(x∉A)}= \{x\in U\mid (x\in A)\land (x\not\in A)\}={x∈U∣(x∈A)∧(x​∈A)}definition of complement
={x∈U∣(x∈A)∧¬(x∈A)}= \{x\in U\mid (x\in A)\land \lnot(x\in A)\}={x∈U∣(x∈A)∧¬(x∈A)}definition of ∉\not\in​∈)
=∅= \emptyset=∅

the last equality following because the proposition (x∈A)∧¬(x∈A)(x\in A)\land \lnot(x\in A)(x∈A)∧¬(x∈A) is false for any xxx. Similarly, we can show that A∪A‾=UA\cup\overline{A}=UA∪A=U and that A‾‾=A\overline{\overline{A}}=AA=A (where A‾‾\overline{\overline{A}}A is the complement of the complement of AAA, that is, the set obtained by taking the complement of A‾\overline{A}A.)

The most important laws for working with complements of sets are De Morgan's Laws for sets. These laws, which follow directly from De Morgan's Laws for logic, state that for any subsets AAA and BBB of a universal set UUU,

A∪B‾=A‾∩B‾\overline{A\cup B}=\overline{A}\cap\overline{B}A∪B=A∩B
A∩B‾=A‾∪B‾\overline{A\cap B}=\overline{A}\cup\overline{B}A∩B=A∪B

For example, we can verify the first of these laws with the calculation

A∪B‾\overline{A\cup B}A∪B={x∈U∣x∉(A∪B)}= \{x\in U\mid x \not\in (A\cup B)\}={x∈U∣x​∈(A∪B)}definition of complement
={x∈U∣¬(x∈A∪B)}= \{x\in U\mid \lnot( x\in A\cup B)\}={x∈U∣¬(x∈A∪B)}definition of ∉\not\in​∈
={x∈U∣¬(x∈A∨x∈B)}= \{x\in U\mid \lnot(x\in A \lor x\in B)\}={x∈U∣¬(x∈A∨x∈B)}definition of ∪\cup∪
={x∈U∣(¬(x∈A))∧(¬(x∈B))}= \{x\in U\mid (\lnot(x\in A))\land(\lnot(x\in B))\}={x∈U∣(¬(x∈A))∧(¬(x∈B))}De Morgan's Law for logic
={x∈U∣(x∉A)∧(x∉B)}= \{x\in U\mid (x\not\in A) \land (x\not\in B)\}={x∈U∣(x​∈A)∧(x​∈B)}definition of ∉\not\in​∈
={x∈U∣(x∈A‾)∧(x∈B‾)}= \{x\in U\mid (x\in\overline{A}) \land (x\in\overline{B})\}={x∈U∣(x∈A)∧(x∈B)}definition of complement
=A‾∩B‾= \overline{A}\cap\overline{B}=A∩Bdefinition of ∩\cap∩

#Comparison with Propositional Boolean Algebra

Just as the laws of logic allow us to do algebra with logical formulas, the laws of set theory allow us to do algebra with sets. Because of the close relationship between logic and set theory, their algebras are very similar. The algebra of sets, like the algebra of logic, is Boolean algebra. When George Boole wrote his 1854 book about logic, it was really as much about set theory as logic. In fact, Boole did not make a clear distinction between a predicate and the set of objects for which that predicate is true. His algebraic laws and formulas apply equally to both cases. More exactly, if we consider only subsets of some given universal set UUU, then there is a direct correspondence between the basic symbols and operations of propositional logic and certain symbols and operations in set theory, as shown in this table:

LogicSet Theory
T\TTUUU

p∨qp\lor qp∨qA∪BA\cup BA∪B
¬p\lnot p¬pA‾\overline{A}A

Any valid logical formula or computation involving propositional variables and the symbols T\TT, F\FF, ∧\land∧, ∨\lor∨, and ¬\lnot¬ can be transformed into a valid formula or computation in set theory by replacing the propositions in the formula with subsets of UUU and replacing the logical symbols with UUU, ∅\emptyset∅, ∩\cap∩, ∪\cup∪, and the complement operator.

Here are some of the laws of Boolean Algebra for sets. AAA, BBB, and CCC are sets. For the laws that involve the complement operator, they are assumed to be subsets of some universal set, UUU. For the most part, these laws correspond directly to laws of Boolean Algebra for propositional logic.

#Precedence

Just as in logic, the operations of set theory can be combined to form complex expressions such as (A∪C)∩(B∪C‾∪D)‾(A\cup C)\cap\overline{(B\cup \overline{C} \cup D)}(A∪C)∩(B∪C∪D)​. Parentheses can always be used in such expressions to specify the order in which the operations are to be performed. In the absence of parentheses, we need precedence rules to determine the order of operation. The precedence rules for the Boolean algebra of sets are carried over directly from the Boolean algebra of propositions. When union and intersection are used together without parentheses, intersection has precedence over union. Furthermore, when several operators of the same type are used without parentheses, then they are evaluated in order from left to right. (Of course, since ∪\cup∪ and ∩\cap∩ are both associative operations, it really doesn't matter whether the order of evaluation is left-to-right or right-to-left.) For example, A∪B∩C∪DA\cup B\cap C \cup DA∪B∩C∪D is evaluated as (A∪((B∩C))∪D(A\cup((B\cap C))\cup D(A∪((B∩C))∪D. The complement operation is a special case. Since it is denoted by drawing a line over its operand, there is never any ambiguity about which part of a formula it applies to.

#Simplification

The laws of set theory can be used to simplify complex expressions involving sets. (As usual, of course, the meaning of "simplification" is partly in the eye of the beholder.) For example, for any sets XXX and YYY,

(X∪Y)∩(Y∪X)(X\cup Y)\cap(Y\cup X)(X∪Y)∩(Y∪X)=(X∪Y)∩(X∪Y)=(X\cup Y)\cap(X\cup Y)=(X∪Y)∩(X∪Y)Commutative Law
=(X∪Y)=(X\cup Y)=(X∪Y)Idempotent Law

where in the second step, the Idempotent Law, which says that A∩A=AA\cap A=AA∩A=A, is applied with A=X∪YA=X\cup YA=X∪Y. For expressions that use the complement operation, it is usually considered to be simpler to apply the operation to an individual set, as in A‾\overline{A}A, rather than to a formula, as in A∩B‾\overline{A\cap B}A∩B. De Morgan's Laws can always be used to simplify an expression in which the complement operation is applied to a formula. For example,

A∩B∪A‾‾A\cap \overline{B\cup\overline{A}}A∩B∪A=A∩(B‾∩A‾‾)= A\cap (\overline{B}\cap\overline{\overline{A}})=A∩(B∩A)De Morgan's Law
=A∩(B‾∩A)= A\cap (\overline{B}\cap A)=A∩(B∩A)Double Complement
=A∩(A∩B‾)= A\cap (A\cap\overline{B})=A∩(A∩B)Commutative Law
=(A∩A)∩B‾)= (A\cap A)\cap \overline{B})=(A∩A)∩B)Associative Law
=A∩B‾= A \cap \overline{B}=A∩BIdempotent Law

As a final example of the relationship between set theory and logic, consider the set-theoretical expression A∩(A∪B)A\cap (A\cup B)A∩(A∪B) and the corresponding compound proposition p∧(p∨q)p\land(p\lor q)p∧(p∨q). (These correspond since for any xxx, x∈A∩(A∪B)≡(x∈A)∧((x∈A)∨(x∈B))x\in A\cap(A\cup B) \equiv (x\in A)\land ((x\in A)\lor (x\in B))x∈A∩(A∪B)≡(x∈A)∧((x∈A)∨(x∈B)).) You might find it intuitively clear that A∩(A∪B)=AA\cap(A\cup B)=AA∩(A∪B)=A. Formally, this follows from the fact that p∧(p∨q)≡pp\land(p\lor q)\equiv pp∧(p∨q)≡p, which might be less intuitively clear and is surprising difficult to prove algebraically from the laws of logic. However, there is another way to check that a logical equivalence is valid: Make a truth table. Consider a truth table for p∧(p∨q)p\land(p\lor q)p∧(p∨q):

pppqqqp∨qp\lor qp∨qp∧(p∨q)p\land (p\lor q)p∧(p∨q)
falsefalsefalsefalse
falsetruetruefalse
truefalsetruetrue
truetruetruetrue

The fact that the first column and the last column of this table are identical shows that p∧(p∨q)≡pp\land(p\lor q)\equiv pp∧(p∨q)≡p. Taking ppp to be the proposition x∈Ax\in Ax∈A and qqq to be the proposition x∈Bx\in Bx∈B, it follows that the sets AAA and A∩(A∪B)A\cap (A\cup B)A∩(A∪B) have the same members and therefore are equal.

#Exercises

  2. Show that for any sets AAA and BBB, A⊆A∪BA\subseteq A\cup BA⊆A∪B and A∩B⊆AA\cap B\subseteq AA∩B⊆A.

  3. Recall that the symbol ⊕\oplus⊕ denotes the logical exclusive or operation. If AAA and BBB sets, define the set A△BA\bigtriangleup BA△B by A△B={x∣(x∈A)⊕(x∈B)}A\bigtriangleup B = \{x\mid (x\in A)\oplus (x\in B)\}A△B={x∣(x∈A)⊕(x∈B)}. Show that A△B=(A∖B)∪(B∖A)A\bigtriangleup B = (A\smallsetminus B)\cup(B\smallsetminus A)A△B=(A∖B)∪(B∖A). (A△BA\bigtriangleup BA△B is known as the symmetric difference of AAA and BBB.)

  4. Let AAA be a subset of some given universal set UUU. Verify that A‾‾=A\overline{\overline{A}}=AA=A and that A∪A‾=UA\cup\overline{A}=UA∪A=U.
