Set Concepts

(Content adapted from Critchlow & Eck)

We deal with the complexity of the world by putting things into categories. There are not just hordes of individual creatures. There are dogs, cats, elephants, and mice. There are mammals, insects, and fish. Animals, vegetables, and minerals. Solids, liquids, and gases. Things that are red. Big cities. Pleasant memories…. Categories build on categories. They are the subject and the substance of thought.

In mathematics, which operates in its own abstract and rigorous world, categories are modeled by sets. A set is just a collection of elements. Along with logic, sets form the "foundation" of mathematics, just as categories are part of the foundation of day-to-day thought. In this chapter, we study sets and relationships among sets.

Basic Concepts#

A set is a collection of elements. A set is defined entirely by the elements that it contains. An element can be anything, including another set. You will notice that this is not a precise mathematical definition. Instead, it is an intuitive description of what the word "set" is supposed to mean: Any time you have a bunch of entities and you consider them as a unit, you have a set. Mathematically, sets are really defined by the operations that can be performed on them. These operations model things that can be done with collections of objects in the real world. These operations are the subject of the branch of mathematics known as set theory.

The most basic operation in set theory is forming a set from a given list of specific entities. The set that is formed in this way is denoted by enclosing the list of entities between a left brace, "{\{{", and a right brace, "}\}}". The entities in the list are separated by commas. For example, the set denoted by

{ 17, π, New York City, Barack Obama, Big Ben }\{\ 17,\ \pi,\ \texttt{New York City},\ \texttt{Barack Obama},\ \texttt{Big Ben}\ \}{ 17, π, New York City, Barack Obama, Big Ben }

is the set that contains the entities 17, π\piπ, New York City, Barack Obama, and Big Ben. These entities are the elements of the set. Since we assume that a set is completely defined by the elements that it contains, the set is well-defined. Of course, we still haven't said what it means to be an "entity." Something as definite as "New York City" should qualify, except that it doesn't seem like New York City really belongs in the world of Mathematics. The problem is that mathematics is supposed to be its own self-contained world, but it is supposed to model the real world. When we use mathematics to model the real world, we admit entities such as New York City and even Big Ben. But when we are doing mathematics per se, we'll generally stick to obviously mathematical entities such as the integer 17 or the real number π\piπ. We will also use letters such as aaa and bbb to refer to entities. For example, when I say something like "Let AAA be the set {a,b,c}\{a,b,c\}{a,b,c}," I mean aaa, bbb, and ccc to be particular, but unspecified, entities.

It's important to understand that a set is defined by the elements that it contains, and not by the order in which those elements might be listed. For example, the notations {a,b,c,d}\{a,b,c,d\}{a,b,c,d} and {b,c,a,d}\{b,c,a,d\}{b,c,a,d} define the same set. Furthermore, a set can only contain one copy of a given element, even if the notation that specifies the set lists the element twice. This means that {a,b,a,a,b,c,a}\{a,b,a,a,b,c,a\}{a,b,a,a,b,c,a} and {a,b,c}\{a,b,c\}{a,b,c} specify exactly the same set. Note in particular that it's incorrect to say that the set {a,b,a,a,b,c,a}\{a,b,a,a,b,c,a\}{a,b,a,a,b,c,a} contains seven elements, since some of the elements in the list are identical. The notation {a,b,c}\{a,b,c\}{a,b,c} can lead to some confusion, since it might not be clear whether the letters aaa, bbb, and ccc are assumed to refer to three different entities. A mathematician would generally not make this assumption without stating it explicitly, so that the set denoted by {a,b,c}\{a,b,c\}{a,b,c} could actually contain either one, two, or three elements. When it is important that different letters refer to different entities, I will say so explicitly, as in "Consider the set {a,b,c}\{a,b,c\}{a,b,c}, where aaa, bbb, and ccc are distinct."

The symbol ∈\in∈ is used to express the relation "is an element of." That is, if aaa is an entity and AAA is a set, then a∈Aa\in Aa∈A is a statement that is true if and only if aaa is one of the elements of AAA. In that case, we also say that aaa is a member of the set AAA. The assertion that aaa is not an element of AAA is expressed by the notation a∉Aa\mathbin{\not\in} Aa​∈A. Note that both a∈Aa\in Aa∈A and ∉A\mathbin{\not\in} A​∈A are statements in the sense of propositional logic. That is, they are assertions which can be either true or false. The statement ∉A\mathbin{\not\in} A​∈A is equivalent to ¬(a∈A)\lnot(a\in A)¬(a∈A).

It is possible for a set to be empty, that is, to contain no elements whatsoever. Since a set is completely determined by the elements that it contains, there is only one set that contains no elements. This set is called the empty set, and it is denoted by the symbol ∅\emptyset∅. Note that for any element aaa, the statement a∈∅a\in\emptyseta∈∅ is false. The empty set, ∅\emptyset∅, can also be denoted by an empty pair of braces, { }\{\ \}{ }.

If AAA and BBB are sets, then, by definition, AAA is equal to BBB if and only if they contain exactly the same elements. In this case, we write A=BA=BA=B. Using the notation of predicate logic, we can say that A=BA=BA=B if and only if ∀x(x∈A↔x∈B)\forall x(x\in A \leftrightarrow x\in B)∀x(x∈A↔x∈B).

Suppose now that AAA and BBB are sets such that every element of AAA is an element of BBB. In that case, we say that AAA is a subset of BBB, i.e. AAA is a subset of BBB if and only if ∀x(x∈A→x∈B)\forall x(x\in A \rightarrow x\in B)∀x(x∈A→x∈B). The fact that AAA is a subset of BBB is denoted by A⊆BA\subseteq BA⊆B. Note that ∅\emptyset∅ is a subset of every set BBB: x∈∅x \in \emptysetx∈∅ is false for any xxx, and so given any BBB, (x∈∅→x∈B)(x\in \emptyset \rightarrow x\in B)(x∈∅→x∈B) is true for all xxx.

If A=BA=BA=B, then it is automatically true that A⊆BA\subseteq BA⊆B and that B⊆AB\subseteq AB⊆A. The converse is also true: If A⊆BA\subseteq BA⊆B and B⊆AB\subseteq AB⊆A, then A=BA=BA=B. This follows from the fact that for any xxx, the statement (x∈A↔x∈B)(x\in A \leftrightarrow x\in B)(x∈A↔x∈B) is logically equivalent to the statement (x∈A→x∈B)∧(x∈B→x∈A)(x\in A\rightarrow x\in B) \land (x\in B\rightarrow x\in A)(x∈A→x∈B)∧(x∈B→x∈A). This fact is important enough to state as a theorem.

Theorem: Set Equality
Let AAA and BBB be sets. Then A=BA=BA=B if and only if both A⊆BA\subseteq BA⊆B and B⊆AB\subseteq AB⊆A.

This theorem expresses the following advice: If you want to check that two sets, AAA and BBB, are equal, you can do so in two steps. First check that every element of AAA is also an element of BBB, and then check that every element of BBB is also an element of AAA.

If A⊆BA\subseteq BA⊆B but A≠BA\not= BA​=B, we say that AAA is a \nw{proper subset} of BBB. We use the notation A⊊BA\varsubsetneq BAB to mean that AAA is a proper subset of BBB. That is, A⊊BA\varsubsetneq BAB if and only if A⊆B∧A≠BA\subseteq B \land A\not= BA⊆B∧A​=B. We will sometimes use A⊇BA\supseteq BA⊇B as an equivalent notation for B⊆AB\subseteq AB⊆A, and A⊋BA\varsupsetneq BAB as an equivalent for B⊊AB\varsubsetneq ABA.

Predicates#

A set can contain an infinite number of elements. In such a case, it is not possible to list all the elements in the set. Sometimes the ellipsis "…" is used to indicate a list that continues on infinitely. For example, N\NN, the set of natural numbers, can be specified as

N={0,1,2,3,… }\N = \{ 0, 1, 2, 3, \dots \}N={0,1,2,3,…}

However, this is an informal notation, which is not really well-defined, and it should only be used in cases where it is clear what it means. It's not very useful to say that "the set of prime numbers is {2,3,5,7,11,13,… }\{2,3,5,7,11,13,\dots\}{2,3,5,7,11,13,…}," and it is completely meaningless to talk about "the set {17,42,105,… }\{17,42,105,\dots\}{17,42,105,…}." Clearly, we need another way to specify sets besides listing their elements. The need is fulfilled by predicates.

If P(x)P(x)P(x) is a predicate, then we can form the set that contains all entities aaa such that aaa is in the domain of discourse for PPP and P(a)P(a)P(a) is true. The notation {x∣P(x)}\{x \mid P(x)\}{x∣P(x)} is used to denote this set. The name of the variable, xxx, is arbitrary, so the same set could equally well be denoted as {z∣P(z)}\{z\mid P(z)\}{z∣P(z)} or {r∣P(r)}\{r\mid P(r)\}{r∣P(r)}. The notation {x∣P(x)}\{x\mid P(x)\}{x∣P(x)} can be read "the set of xxx such that P(x)P(x)P(x)." For example, if E(x)E(x)E(x) is the predicate "xxx is an even number," and if the domain of discourse for EEE is the set N\NN of natural numbers, then the notation {x∣E(x)}\{x\mid E(x)\}{x∣E(x)} specifies the set of even natural numbers. That is,

{x∣E(x)}={0,2,4,6,8,… }\{x\mid E(x)\} = \{0,2,4,6,8,\dots\}{x∣E(x)}={0,2,4,6,8,…}

It turns out, for deep and surprising reasons that we will discuss later in this section, that we have to be a little careful about what counts as a predicate. In order for the notation {x∣P(x)}\{x\mid P(x)\}{x∣P(x)} to be valid, we have to assume that the domain of discourse of PPP is in fact a set. (You might wonder how it could be anything else. That's the surprise!) Often, it is useful to specify the domain of discourse explicitly in the notation that defines a set. In the above example, to make it clear that xxx must be a natural number, we could write the set as {x∈N∣E(x)}\{x\in\N \mid E(x)\}{x∈N∣E(x)}. This notation can be read as "the set of all xxx in N\NN such that E(x)E(x)E(x)." More generally, if XXX is a set and PPP is a predicate whose domain of discourse includes all the elements of XXX, then the notation

{x∈X∣P(x)}\{x\in X\mid P(x)\}{x∈X∣P(x)}

is the set that consists of all entities aaa that are members of the set XXX and for which P(a)P(a)P(a) is true. In this notation, we don't have to assume that the domain of discourse for PPP is a set, since we are effectively limiting the domain of discourse to the set XXX. The set denoted by {x∈X∣P(x)}\{x\in X\mid P(x)\}{x∈X∣P(x)} could also be written as {x∣x∈X∧P(x)}\{x\mid x\in X\land P(x)\}{x∣x∈X∧P(x)}.

We can use this notation to define the set of prime numbers in a rigorous way. A prime number is a natural number nnn which is greater than 1 and which satisfies the property that for any factorization n=xyn=xyn=xy, where xxx and yyy are natural numbers, either xxx or yyy must be nnn. We can express this definition as a predicate and define the set of prime numbers as

{n∈N∣(n>1)∧∀x∀y((x∈N∧y∈N∧n=xy)→(x=n∨y=n))}.\{n\in\N\mid (n>1)\land\forall x\forall y\big((x\in\N\land y\in\N\land n=xy)\rightarrow(x=n\lor y=n)\big)\}.{n∈N∣(n>1)∧∀x∀y((x∈N∧y∈N∧n=xy)→(x=n∨y=n))}.

Admittedly, this definition is hard to take in in one gulp. But this example shows that it is possible to define complex sets using predicates.

Operations#

Now that we have a way to express a wide variety of sets, we turn to operations that can be performed on sets. The most basic operations on sets are union and intersection. If AAA and BBB are sets, then we define the union of AAA and BBB to be the set that contains all the elements of AAA together with all the elements of BBB. The union of AAA and BBB is denoted by A∪BA\cup BA∪B. The union can be defined formally as

A∪B={x∣x∈A∨x∈B}.A\cup B = \{x\mid x\in A \lor x\in B\}.A∪B={x∣x∈A∨x∈B}.

The intersection of AAA and BBB is defined to be the set that contains every entity that is both a member of AAA and a member of BBB. The intersection of AAA and BBB is denoted by A∩BA\cap BA∩B. Formally,

A∩B={x∣x∈A∧x∈B}.A\cap B = \{x\mid x\in A \land x\in B\}.A∩B={x∣x∈A∧x∈B}.

An entity gets into A∪BA\cup BA∪B if it is in either AAA or BBB. It gets into A∩BA\cap BA∩B if it is in both AAA and BBB. Note that the symbol for the logical "or" operator, ∨\lor∨, is similar to the symbol for the union operator, ∪\cup∪, while the logical "and" operator, ∧\land∧, is similar to the intersection operator, ∩\cap∩.

The set difference of two sets, AAA and BBB, is defined to be the set of all entities that are members of AAA but are not members of BBB. The set difference of AAA and BBB is denoted A∖BA\smallsetminus BA∖B. The idea is that A∖BA\smallsetminus BA∖B is formed by starting with AAA and then removing any element that is also found in BBB. Formally,

A∖B={x∣x∈A∧x∉B}.A\smallsetminus B = \{x\mid x\in A \land x\mathbin{\not\in} B\}.A∖B={x∣x∈A∧x​∈B}.

Union and intersection are clearly commutative operations. That is, A∪B=B∪AA\cup B=B\cup AA∪B=B∪A and A∩B=B∩AA\cap B=B\cap AA∩B=B∩A for any sets AAA and BBB. However, set difference is not commutative. In general, A∖B≠B∖AA\smallsetminus B \not= B\smallsetminus AA∖B​=B∖A.

Suppose that A={a,b,c}A=\{a,b,c\}A={a,b,c}, that B={b,d}B=\{b,d\}B={b,d}, and that C={d,e,f}C=\{d,e,f\}C={d,e,f}. Then we can apply the definitions of union, intersection, and set difference to compute, for example, that:

A∪B={a,b,c,d}A\cup B = \{a,b,c,d\}A∪B={a,b,c,d}A∩B={b}A\cap B = \{b\}A∩B={b}A∖B={a,c}A\smallsetminus B = \{a,c\}A∖B={a,c}
A∪C={a,b,c,d,e,f}A\cup C = \{a,b,c,d,e,f\}A∪C={a,b,c,d,e,f}A∩C=∅A\cap C = \emptysetA∩C=∅A∖C={a,b,c}A\smallsetminus C = \{a,b,c\}A∖C={a,b,c}

In this example, the sets AAA and CCC have no elements in common, so that A∩C=∅A\cap C=\emptysetA∩C=∅. There is a term for this: Two sets are said to be disjoint if they have no elements in common. That is, for any sets AAA and BBB, AAA and BBB are said to be disjoint if and only if A∩B=∅A\cap B=\emptysetA∩B=∅.

Of course, the set operations can also be applied to sets that are defined by predicates. For example, let L(x)L(x)L(x) be the predicate "xxx is lucky," and let W(x)W(x)W(x) be the predicate "xxx is wise," where the domain of discourse for each predicate is the set of people. Let X={x∣L(x)}X = \{x\mid L(x)\}X={x∣L(x)}, and let Y={x∣W(x)}Y=\{x\mid W(x)\}Y={x∣W(x)}. Then

X∪Y={x∣L(x)∨W(x)}={people who are lucky or wise}X\cup Y = \{x\mid L(x)\lor W(x)\} = \{\text{people who are lucky or wise}\}X∪Y={x∣L(x)∨W(x)}={people who are lucky or wise}
X∩Y={x∣L(x)∧W(x)}={people who are lucky and wise}X\cap Y = \{x\mid L(x)\land W(x)\} = \{\text{people who are lucky and wise}\}X∩Y={x∣L(x)∧W(x)}={people who are lucky and wise}
X∖Y={x∣L(x)∧¬W(x)}={people who are lucky but not wise}X\smallsetminus Y = \{x\mid L(x)\land \lnot W(x)\} = \{\text{people who are lucky but not wise}\}X∖Y={x∣L(x)∧¬W(x)}={people who are lucky but not wise}
Y∖X={x∣W(x)∧¬L(x)}={people who are wise but not lucky}Y\smallsetminus X = \{x\mid W(x)\land \lnot L(x)\} = \{\text{people who are wise but not lucky}\}Y∖X={x∣W(x)∧¬L(x)}={people who are wise but not lucky}

You have to be a little careful with the English word "and." We might say that the set X∪YX\cup YX∪Y contains people who are lucky and people who are wise. But what this means is that a person gets into the set X∪YX\cup YX∪Y either by being lucky or by being wise, so X∪YX\cup YX∪Y is defined using the logical "or" operator, ∨\lor∨.

Sets can contain other sets as elements. For example, the notation {a,{b}}\{a,\{b\}\}{a,{b}} defines a set that contains two elements, the entity aaa and the set {b}\{b\}{b}. Since the set {b}\{b\}{b} is a member of the set {a,{b}}\{a,\{b\}\}{a,{b}}, we have that {b}∈{a,{b}}\{b\}\in\{a,\{b\}\}{b}∈{a,{b}}. On the other hand, provided that a≠ba\not=ba​=b, the statement {b}⊆{a,{b}}\{b\}\subseteq\{a,\{b\}\}{b}⊆{a,{b}} is false, since saying {b}⊆{a,{b}}\{b\}\subseteq\{a,\{b\}\}{b}⊆{a,{b}} is equivalent to saying that b∈{a,{b}}b\in\{a,\{b\}\}b∈{a,{b}}, and the entity bbb is not one of the two members of {a,{b}}\{a,\{b\}\}{a,{b}}. For the entity aaa, it is true that {a}⊆{a,{b}}\{a\}\subseteq\{a,\{b\}\}{a}⊆{a,{b}}.

Given a set AAA, we can construct the set that contains all the subsets of AAA. This set is called the power set of AAA, and is denoted P(A){\mathscr P}(A)P(A). Formally, we define

P(A)={X∣X⊆A}.{\mathscr P}(A)=\{X\mid X\subseteq A\}.P(A)={X∣X⊆A}.

For example, if A={a,b}A=\{a,b\}A={a,b}, then the subsets of AAA are the empty set, {a}\{a\}{a}, {b}\{b\}{b}, and {a,b}\{a,b\}{a,b}, so the power set of AAA is set given by

P(A)={ ∅, {a}, {b}, {a,b} }.{\mathscr P}(A) = \{\,\emptyset,\,\{a\},\,\{b\},\,\{a,b\}\,\}.P(A)={∅,{a},{b},{a,b}}.

Note that since the empty set is a subset of any set, the empty set is an element of the power set of any set. That is, for any set AAA, ∅⊆A\emptyset\subseteq A∅⊆A and ∅∈P(A)\emptyset\in{\mathscr P}(A)∅∈P(A). Since the empty set is a subset of itself, and is its only subset, we have that P(∅)={∅}{\mathscr P}(\emptyset) = \{\emptyset\}P(∅)={∅}. The set {∅}\{\emptyset\}{∅} is not empty. It contains one element, namely ∅\emptyset∅.

Russell's Paradox#

We remarked earlier in this section that the notation {x∣P(x)}\{x \mid P(x)\}{x∣P(x)} is only valid if the domain of discourse of PPP is a set. This might seem a rather puzzling thing to say—after all, why and how would the domain of discourse be anything else? The answer is related to Russell's Paradox, which shows that it is logically impossible for the set of all sets to exist. This impossibility can be demonstrated using a proof by contradiction. In the proof, we use the existence of the set of all sets to define another set which cannot exist because its existence would lead to a logical contradiction.

Theorem: Russell
There is no set of all sets.

Proof Suppose that the set of all sets exists. We will show that this assumption leads to a contradiction. Let VVV be the set of all sets. We can then define the set RRR to be the set which contains every set that does not contain itself. That is,

R={X∈V∣X∉X}R=\{X\in V\mid X\mathbin{\not\in} X\}R={X∈V∣X​∈X}

Now, we must have either R∈RR\in RR∈R or R∉RR\mathbin{\not\in} RR​∈R. We will show that either case leads to a contradiction.

Consider the case where R∈RR\in RR∈R. Since R∈RR\in RR∈R, RRR must satisfy the condition for membership in RRR. A set XXX is in RRR iff X∉XX\mathbin{\not\in} XX​∈X. To say that RRR satisfies this condition means that R∉RR\mathbin{\not\in} RR​∈R. That is, from the fact that R∈RR\in RR∈R, we deduce the contradiction that R∉RR\mathbin{\not\in} RR​∈R.

Now consider the remaining case, where R∉RR\mathbin{\not\in} RR​∈R. Since R∉RR\mathbin{\not\in} RR​∈R, RRR does not satisfy the condition for membership in RRR. Since the condition for membership is that R∉RR\mathbin{\not\in} RR​∈R, and this condition is false, the statement R∉RR\mathbin{\not\in} RR​∈R must be false. But this means that the statement R∈RR\in RR∈R is true. From the fact that R∉RR\mathbin{\not\in} RR​∈R, we deduce the contradiction that R∈RR\in RR∈R.

Since both possible cases, R∈RR\in RR∈R and R∉RR\mathbin{\not\in} RR​∈R, lead to contradictions, we see that it is not possible for RRR to exist. Since the existence of RRR follows from the existence of VVV, we see that VVV also cannot exist.

To avoid Russell's paradox, we must put limitations on the construction of new sets. We can't force the set of all sets into existence simply by thinking of it. We can't form the set {x∣P(x)}\{x\mid P(x)\}{x∣P(x)} unless the domain of discourse of PPP is a set. Any predicate QQQ can be used to form a set {x∈X∣Q(x)}\{x\in X\mid Q(x)\}{x∈X∣Q(x)}, but this notation requires a pre-existing set XXX. Predicates can be used to form subsets of existing sets, but they can't be used to form new sets completely from scratch.

The notation {x∈A∣P(x)}\{x\in A\mid P(x)\}{x∈A∣P(x)} is a convenient way to effectively limit the domain of discourse of a predicate, PPP, to members of a set, AAA, that we are actually interested in. We will use a similar notation with the quantifiers ∀\forall∀ and ∃\exists∃. The proposition (∀x∈A)(P(x))(\forall x\in A)(P(x))(∀x∈A)(P(x)) is true if and only if P(a)P(a)P(a) is true for every element aaa of the set AAA. And the proposition (∃x∈A)(P(x))(\exists x\in A)(P(x))(∃x∈A)(P(x)) is true if and only if there is some element aaa of the set AAA for which P(a)P(a)P(a) is true. These notations are valid only when AAA is contained in the domain of discourse for PPP. As usual, we can leave out parentheses when doing so introduces no ambiguity. So, for example, we might write ∀x∈A  P(x)\forall x\in A\;P(x)∀x∈AP(x).

Exercises#

  1. If we don't make the assumption that aaa, bbb, and ccc are distinct, then the set denoted by {a,b,c}\{a,b,c\}{a,b,c} might actually contain either 1, 2, or 3 elements. How many different elements might the set { a, b, {a}, {a,c}, {a,b,c} }\{\,a,\,b,\,\{a\},\,\{a,c\},\,\{a,b,c\}\,\}{a,b,{a},{a,c},{a,b,c}} contain? Explain your answer.

    Answer

    If they are all distinct, then there are five elements: aaa, bbb, and the sets {a}\{a\}{a}, {a,c}\{a,c\}{a,c}, and {a,b,c}\{a,b,c\}{a,b,c}. If aaa and bbb are the same, then there are three elements: aaa, {a}\{a\}{a}, and {a,c}\{a,c\}{a,c} (which is the same as {a,b,c}\{a,b,c\}{a,b,c}). If aaa and ccc are the same, then there are four elements: aaa, bbb, {a}\{a\}{a}, and {a,b}\{a,b\}{a,b}. If bbb and ccc are the same, then we actually get the same four elements as when aaa and ccc are the same (this should make sense, because if ccc is the same as either of the others, then it is redundant in both of the places it occurs). Finally, if all three are the same, then we have just two elements: aaa and {a}\{a\}{a}.

      A∪B=BA\cup B=BA∪B=B, A∩B=AA\cap B=AA∩B=A, A∖B=∅A\smallsetminus B=\emptysetA∖B=∅

  3. Recall that N\NN represents the set of natural numbers. That is, N={0,1,2,3,… }\N=\{0,1,2,3,\dots\}N={0,1,2,3,…}.   4. Find P({1,2,3}){\mathscr P}\big(\{1,2,3\}\big)P({1,2,3}). (It has eight elements.)

  5. Assume that aaa and bbb are entities and that a≠ba\not=ba​=b. Let AAA and BBB be the sets defined by A={ a, {b}, {a,b} }A=\{\,a,\,\{b\},\,\{a,b\}\,\}A={a,{b},{a,b}} and B={ a, b, {a,{b}} }B=\{\,a,\,b,\,\{a,\{b\}\}\,\}B={a,b,{a,{b}}}. Determine whether each of the following statements is true or false. Explain your answers.

    • b∈Ab\in Ab∈A
      Answer

      False. The set {b}\{b\}{b} is an element of AAA, but not the entity bbb itself.

    • {a,b}⊆A\{a,b\}\subseteq A{a,b}⊆A
      Answer

      False, since b∉Ab\not\in Ab​∈A.

    • {a,b}⊆B\{a,b\}\subseteq B{a,b}⊆B
      Answer

      True, since both aaa and bbb are elements of BBB.

    • {a,b}∈B\{a,b\}\in B{a,b}∈B
      Answer

      False. The set {a,b}\{a,b\}{a,b} is distinct from each of the three elements of BBB.

    • {a,{b}}∈A\{a,\{b\}\}\in A{a,{b}}∈A
      Answer

      False. Although {a,{b}}⊆A\{a,\{b\}\}\subseteq A{a,{b}}⊆A, it is not an element of AAA.

    • {a,{b}}∈B\{a,\{b\}\}\in B{a,{b}}∈B
      Answer

      True. It is one of the elements of BBB, along with aaa and bbb.

  6. Since P(A){\mathscr P}(A)P(A) is a set, it is possible to form the set P(P(A)){\mathscr P}\big({\mathscr P}(A)\big)P(P(A)). What is P(P(∅)) {\mathscr P}\big({\mathscr P}(\emptyset)\big)\,P(P(∅))? What is P(P({a,b})) {\mathscr P}\big({\mathscr P}(\{a,b\})\big)\,P(P({a,b}))? (It has sixteen elements.)

  7. In the English sentence, "She likes men who are tall, dark, and handsome," does she like an intersection or a union of sets of men? How about in the sentence, "She likes men who are tall, men who are dark, and men who are handsome?"

    Answer

    In the first sentence it is an intersection: she likes the men who are in all three of the sets "tall", "dark", and "handsome". In the second sentence it is a union: she likes men in the "tall" set, men in the "dark" set, and men in the "handsome" set.

  8. If AAA is any set, what can you say about A∪A A\cup A\,A∪A? About A∩A A\cap A\,A∩A? About A∖A A\smallsetminus A\,A∖A? Why?

    Answer

    A∪A=AA\cup A=AA∪A=A, since adding the elements of AAA to itself doesn't add anything new. A∩A=AA\cap A=AA∩A=A as well, since they have every element of AAA in common. A∖A=∅A\smallsetminus A=\emptysetA∖A=∅, since removing all of the elements of AAA from AAA leaves nothing.

  9. Suppose that AAA and BBB are sets such that A⊆BA\subseteq BA⊆B. What can you say about A∪B A\cup B\,A∪B? About A∩B A\cap B\,A∩B? About A∖B A\smallsetminus B\,A∖B? Why?

    Answer

    A∪B=BA\cup B=BA∪B=B, since everything in AAA is already in BBB. A∩B=AA\cap B=AA∩B=A, since every element of AAA is also in BBB. A∖B=∅A\smallsetminus B=\emptysetA∖B=∅, since every element of AAA will be removed.

  10. Suppose that AAA, BBB, and CCC are sets. Show that C⊆A∩BC\subseteq A\cap BC⊆A∩B if and only if (C⊆A)∧(C⊆B)(C\subseteq A)\land (C\subseteq B)(C⊆A)∧(C⊆B).

    Answer

    If C⊆A∩BC\subseteq A\cap BC⊆A∩B, then for each c∈Cc\in Cc∈C we know that c∈{x∣x∈A∧x∈B}c\in\{x\mid x\in A\land x\in B\}c∈{x∣x∈A∧x∈B}, which means c∈Ac\in Ac∈A and c∈Bc\in Bc∈B. Since this is true for every c∈Cc\in Cc∈C, we may conclude C⊆AC\subseteq AC⊆A and C⊆BC\subseteq BC⊆B.

    Conversely, suppose that (C⊆A)∧(C⊆B)(C\subseteq A)\land (C\subseteq B)(C⊆A)∧(C⊆B). That means that for each c∈Cc\in Cc∈C, we know that c∈Ac\in Ac∈A and c∈Bc\in Bc∈B, which is enough to establish that c∈{x∣x∈A∧x∈B}c\in\{x\mid x\in A\land x\in B\}c∈{x∣x∈A∧x∈B}. Therefore, C⊆A∩BC\subseteq A\cap BC⊆A∩B.

  11. Suppose that AAA, BBB, and CCC are sets, and that A⊆BA\subseteq BA⊆B and B⊆CB\subseteq CB⊆C. Show that A⊆CA\subseteq CA⊆C.

    Answer

    If A⊆BA\subseteq BA⊆B and B⊆CB\subseteq CB⊆C, then take any x∈Ax\in Ax∈A. We know that x∈Bx\in Bx∈B because AAA is a subset of BBB; furthermore, we also find that x∈Cx\in Cx∈C because BBB is a subset of CCC. Therefore, for every x∈Ax\in Ax∈A we also have x∈Cx\in Cx∈C, hence A⊆CA\subseteq CA⊆C.

  12. Suppose that AAA and BBB are sets such that A⊆BA\subseteq BA⊆B. Is it necessarily true that P(A)⊆P(B) {\mathscr P}(A)\subseteq {\mathscr P}(B)\,P(A)⊆P(B)? Why or why not?

    Answer

    If x∈P(A)x\in{\mathscr P}(A)x∈P(A), then by the definition of the powerset we know that x⊆Ax\subseteq Ax⊆A. Since A⊆BA\subseteq BA⊆B, we know by the previous exercise that x⊆Bx\subseteq Bx⊆B, therefore we also have x∈P(B)x\in{\mathscr P}(B)x∈P(B). Since this is true for arbitrary xxx, we find that P(A)⊆P(B){\mathscr P}(A)\subseteq {\mathscr P}(B)P(A)⊆P(B) is true.
