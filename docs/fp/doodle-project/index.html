DPoodle Graphics Drawing Project

For Monday, March 30, create a drawing using the DPoodle graphics library. You should feel free to be as creative as you like, the only requirement is that the drawing use recursion in an essential way.

Here is an editor where you can enter your code. Be sure to copy and paste it somewhere else because this editor will be cleared when the page reloads. I suggest keeping a copy in a text editor such as WordPad or TextEdit. When you are done, upload it to Moodle: Section A or Section B.
