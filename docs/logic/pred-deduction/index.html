Natural Deduction for Predicate Logic

Before seeing the natural deduction rules for handling the quantifiers (∀\forall∀ and ∃\exists∃), we will need to be careful about how we handle variables in proofs. We will now have three1 different uses of identifiers in our proofs, representing different entities that we want to be able to refer to by name:

  • Propositions (ppp, qqq, …) and predicates (PPP, QQQ, …) stand for primitive statements that may be true or false, perhaps depending on the values of arguments. When we show that an argument is valid for ppp and Q(a)Q(a)Q(a), for example, we have also shown that the argument is valid when those propositional and predicate variables are replaced by actual statements, such as "It is raining" or "aaa is an even number". As long as we are consistent about replacing variables with the same statement each time they occur, the resulting argument will have the same structure and be equally valid. If we are careful, we can even perform substitutions such as ppp becomes qqq and (simultaneously) qqq becomes ¬p\lnot p¬p; for example, if we have proven the argument p→q,¬q⊢¬pp\rightarrow q, \lnot q\vdash\lnot pp→q,¬q⊢¬p (Modus Tollens), then applying this substitution would allow us to also use the argument q→¬p,¬¬p⊢¬qq\rightarrow\lnot p, \lnot\lnot p\vdash\lnot qq→¬p,¬¬p⊢¬q as a step in a proof.

  • Line labels (iii, jjj, …, as well as ℓ1\ell_1ℓ1​, ℓ2\ell_2ℓ2​, …) allow us to name individual steps and assumptions in our proofs. I have been following the convention that lines in an actual proof are named ℓ1\ell_1ℓ1​, ℓ2\ell_2ℓ2​, …, although they could be given more meaningful names such as hypothesis, first_case, or premise2. When describing the deduction rules, I have been using labels such as iii and jjj to emphasize that the line being referenced could be anywhere earlier in the proof (as long as it is not local to a nested subproof).

  • Entity variables (xxx, yyy, …, as well as aaa, bbb, …) stand for entities from the domain of discourse. I have been using aaa, bbb, … to stand for particular values, such as the number 42 or the student Joe, that might be used in an argument, so Q(a)Q(a)Q(a) could represent the statement "42 is an even number." When the variable is bound by a quantifier, I generally use xxx or yyy instead, to emphasize that the formula is saying something about the full range of values from the domain of discourse, and we are not free to interpret it as any particular entity. The actual name of a bound variable does not matter, only the association between its uses and the quantifier that is binding it. That is, we will consider ∃xP(x)\exists xP(x)∃xP(x) and ∃yP(y)\exists yP(y)∃yP(y) to be the same (and we may choose the name of the variable to avoid confusion with other variables as needed).2

Just as we can introduce temporary assumptions into subproofs, that are only available within that nested block of the proof, we will also introduce temporary entity variables in proofs by writing an assumption such as "fresh x0x_0x0​" (or x1x_1x1​, …, as needed). This means that, within that subproof, the variable x0x_0x0​ will refer to some entity from the domain of discourse. Unlike the variables (xxx, yyy, …) bound by quantifiers, such a fresh variable should be understood as representing a single entity; it can stand alone in a statement such as Q(x0)Q(x_0)Q(x0​). However, unlike the top-level free variables (aaa, bbb, …), we do not get to choose a value for a fresh variable, nor is it allowed to escape from its subproof. That is, no formula outside of the braces is allowed to refer to x0x_0x0​.

#Universal Quantification

To prove a proposition ∀xP(x)\forall xP(x)∀xP(x), and "introduce" a universal quantifier, we need to show that P(x)P(x)P(x) is true regardless of which entity is substituted for xxx. This is what a subproof with a fresh variable gives us:

i:fresh x0⇒{…P(x0)}∴∀xP(x),∀I i\begin{array}{l} i: \text{fresh}\ x_0\Rightarrow\{\\ \quad\ldots\\ \quad P(x_0)\\ \}\\ \hline\therefore \forall xP(x), \quad\forall I\ i \end{array}i:fresh x0​⇒{…P(x0​)}∴∀xP(x),∀I i​​

The nested proof is a demonstration that P(x0)P(x_0)P(x0​) is true with no external assumptions on which entity x0x_0x0​ refers to. The only way we can do this is if P(x)P(x)P(x) is actually true for all entities xxx in the domain of discourse, that is, ∀xP(x)\forall xP(x)∀xP(x).

Here is an example, proving the validity of the argument p⊢∀x(Q(x)→p∧Q(x))p\vdash\forall x(Q(x)\rightarrow p\land Q(x))p⊢∀x(Q(x)→p∧Q(x)):

At step ℓ5\ell_5ℓ5​, we are asserting that Q(x0)→p∧Q(x0)Q(x_0)\rightarrow p\land Q(x_0)Q(x0​)→p∧Q(x0​) is true for some entity x0x_0x0​, but since we were able to conclude this with no extra conditions on which entity that is, in ℓ6\ell_6ℓ6​ we can introduce the universal quantifier. At this point, there are no remaining references to x0x_0x0​, so nothing about the result can depend on which entity we use—it must be true for all of them.

The corresponding elimination rule for the universal quantifier is simple: from the premise ∀xP(x)\forall xP(x)∀xP(x) we may conclude that the predicate PPP holds for any entity aaa in the domain of discourse:

i:∀xP(x)∴P(a),∀E i,a\begin{array}{l} i: \forall xP(x)\\ \hline\therefore P(a), \quad\forall E\ i, a \end{array}i:∀xP(x)∴P(a),∀E i,a​​

For example, we can use this to prove the following quantified version of modus ponens: ∀x(P(x)→Q(x)),P(a)⊢Q(a)\forall x(P(x)\rightarrow Q(x)), P(a)\vdash Q(a)∀x(P(x)→Q(x)),P(a)⊢Q(a). That is, if P(x)P(x)P(x) implies Q(x)Q(x)Q(x) for each xxx, then knowing that P(a)P(a)P(a) holds for some aaa lets us conclude that also Q(a)Q(a)Q(a) holds. Here is the proof:

#Existential Quantification

Extending the analogy between the conjunction/disjunction operators and the universal/existential quantifiers, we find that the rule for introducing an existential is the dual of the universal elimination rule:

i:P(a)∴∃xP(x),∃I i,a\begin{array}{l} i: P(a)\\ \hline\therefore \exists xP(x), \quad\exists I\ i, a \end{array}i:P(a)∴∃xP(x),∃I i,a​​

That is, to show that there exists some xxx making P(x)P(x)P(x) true it is enough to show a specific aaa in the domain of discourse such that P(a)P(a)P(a) holds.

For example, we may prove the validity of the argument ∀xP(x)⊢∃xP(x)\forall xP(x)\vdash\exists xP(x)∀xP(x)⊢∃xP(x), as long as we are able to name some element aaa of the domain of discourse:

ℓ1:∀xP(x)premiseℓ2:P(a)∀E ℓ1,aℓ3:∃xP(x)∃I ℓ2,a\begin{array}{l|l} \ell_1: \forall xP(x) & \text{premise}\\ \ell_2: P(a) & \forall E\ \ell_1, a\\ \ell_3: \exists xP(x) & \exists I\ \ell_2, a \end{array}ℓ1​:∀xP(x)ℓ2​:P(a)ℓ3​:∃xP(x)​premise∀E ℓ1​,a∃I ℓ2​,a​

For a different sort of example, suppose we are proving statements of arithmetic, and we have some means of deriving various equations and inequations about integers. Then we might see the following in a proof:

ℓ1:6⋅7=42arithmeticℓ2:7>1arithmeticℓ3:7>1∧6⋅7=42∧I ℓ2,ℓ1ℓ4:∃n(n>1∧6⋅n=42)∃I ℓ3,7ℓ5:6>1arithmeticℓ6:6>1∧∃n(n>1∧6⋅n=42)∧I ℓ5,ℓ4ℓ7:∃m(m>1∧∃n(n>1∧m⋅n=42))∃I ℓ6,6\begin{array}{l|l} \ell_1: 6\cdot 7=42 & \text{arithmetic}\\ \ell_2: 7>1 & \text{arithmetic}\\ \ell_3: 7>1 \land 6\cdot 7=42 & \land I\ \ell_2, \ell_1\\ \ell_4: \exists n(n>1 \land 6\cdot n=42) & \exists I\ \ell_3, 7\\ \ell_5: 6>1 & \text{arithmetic}\\ \ell_6: 6>1 \land \exists n(n>1 \land 6\cdot n=42) & \land I\ \ell_5, \ell_4\\ \ell_7: \exists m(m>1 \land \exists n(n>1 \land m\cdot n=42)) & \exists I\ \ell_6, 6 \end{array}ℓ1​:6⋅7=42ℓ2​:7>1ℓ3​:7>1∧6⋅7=42ℓ4​:∃n(n>1∧6⋅n=42)ℓ5​:6>1ℓ6​:6>1∧∃n(n>1∧6⋅n=42)ℓ7​:∃m(m>1∧∃n(n>1∧m⋅n=42))​arithmeticarithmetic∧I ℓ2​,ℓ1​∃I ℓ3​,7arithmetic∧I ℓ5​,ℓ4​∃I ℓ6​,6​

The conclusion on the last line shows that the number 42 is composite. There are other proofs of this fact (for example, starting from 3⋅14=423\cdot 14=423⋅14=42); to show that 42 is composite, we only have to come up with one way of factoring it into numbers greater than 1.

Finally, just as with the elimination rule for disjunction, the elimination rule for the existential is somewhat more complicated. However, instead of doing a case analysis of the two ways that p∨qp\lor qp∨q could be true, and show that we can conclude a common result rrr in either case, for the existential we must analyze the potentially infinite number of cases (over all of the entities in our domain of discourse) and show that in each case we may conclude the same common result. The idea of conducting a subproof with extra assumptions, including a fresh variable to stand for whatever entity it is that "witnesses" the truth of the existential, captures what we need here:

i:∃xP(x)j:fresh x0,P(x0)⇒{…r}∴r,∃E i,j\begin{array}{l} i: \exists xP(x)\\ j: \text{fresh}\ x_0, P(x_0)\Rightarrow\{\\ \quad\ldots\\ \quad r\\ \}\\ \hline\therefore r, \quad\exists E\ i, j \end{array}i:∃xP(x)j:fresh x0​,P(x0​)⇒{…r}∴r,∃E i,j​​

In the nested proof, we assume two additional pieces of information: first, that x0x_0x0​ is the name of a value in the domain of discourse, and second, that we have the additional fact that P(x0)P(x_0)P(x0​) is true. Within the subproof, we may use this extra knowledge to prove our conclusion rrr. However, since x0x_0x0​ needs to be "fresh" to the subproof, the identity of the witnessing entity is not allowed to escape from the block3 This meshes with our understanding of the existential—it tells us that there is some xxx making P(x)P(x)P(x) true, but it doesn't tell us which entity it is.

We may now prove a law about the interaction of the existential and conjunction: ∃x(P(x)∧Q(x))⊢(∃xP(x))∧(∃xQ(x))\exists x(P(x)\land Q(x))\vdash(\exists xP(x))\land(\exists xQ(x))∃x(P(x)∧Q(x))⊢(∃xP(x))∧(∃xQ(x)).

It is instructive to see where the proof attempt fails if we try to go the other direction and show (∃xP(x))∧(∃xQ(x))⊢∃x(P(x)∧Q(x))(\exists xP(x))\land(\exists xQ(x))\vdash\exists x(P(x)\land Q(x))(∃xP(x))∧(∃xQ(x))⊢∃x(P(x)∧Q(x)):

The problem comes in step ℓ5\ell_5ℓ5​, where we must introduce a different variable, x1x_1x1​, as the witness for ∃xQ(x)\exists xQ(x)∃xQ(x), because of the requirement that the variable be fresh. This should match your intuition for why the claim failed: although there exists an xxx making P(x)P(x)P(x) true and there also exists an xxx making Q(x)Q(x)Q(x) true, there is no reason to believe that they are the same entities. Indeed, if P(x)P(x)P(x) says "xxx is odd" and Q(x)Q(x)Q(x) says "xxx is even", then there will be no overlap in the entities making each true. There are odd numbers, and there are even numbers, but there are no numbers that are both odd and even.
