Boolean Algebra of Sets

(Content adapted from Critchlow & Eck)

It is clear that set theory is closely related to logic. The intersection\index{intersection} and union\index{union} of sets can be defined in terms of the logical and''\index{and (logical operator)} and logical or''\index{or (logical operator)} operators. The notation {x\stP(x)}\{x\st P(x)\}{x\stP(x)} makes it possible to use predicates to specify sets. And if AAA is any set, then the formula x∈Ax\in Ax∈A defines a one place predicate that is true for an entity xxx if and only if xxx is a member of~AAA. So it should not be a surprise that many of the rules of logic have analogs in set theory.

For example, we have already noted that ∪\cup∪ and ∩\cap∩ are commutative operations. This fact can be verified using the rules of logic. Let AAA and BBB be sets. According to the definition of equality of sets, we can show that A∪B=B∪AA\cup B=B\cup AA∪B=B∪A by showing that ∀x ((x∈A∪B)\IFF(x∈B∪A))\forall x\,\big((x\in A\cup B)\IFF(x\in B\cup A)\big)∀x((x∈A∪B)\IFF(x∈B∪A)). But for any xxx, \begin{align} x\in A\cup B &\IFF x\in A \OR x\in B &&\text{(definition of ∪\cup∪)}\ &\IFF x\in B \OR x\in A &&\text{(commutativity of \OR\OR\OR)}\ &\IFF x\in B \cup A &&\text{(definition of ∪\cup∪)} \end{align} The commutativity of ∩\cap∩ follows in the same way from the definition of ∩\cap∩ in terms of~\AND\AND\AND and the commutativity of~\AND\AND\AND, and a similar argument shows that union and intersection are associative operations.

The distributive laws for propositional logic give rise to two similar rules in set theory. Let AAA, BBB, and CCC be any sets. Then \begin{align} A\cup(B\cap C)&=(A\cup B)\cap(A\cup C)\ \intertext{and} A\cap(B\cup C)&=(A\cap B)\cup(A\cap C) \end{align} These rules are called the distributive laws\index{distributive law} for set theory. To verify the first of these laws, we just have to note that for any~xxx, \begin{align} x\in A&\cup(B\cap C) \ &\IFF (x\in A)\OR((x\in B)\AND (x\in C))
&&\text{(definition of ∪\cup∪, ∩\cap∩)}\ &\IFF ((x\in A)\OR(x\in B)) \AND((x \in A)\OR (x\in C)) &&\text{(distributivity of \OR\OR\OR)}\ &\IFF (x\in A\cup B) \AND (x \in A\cup C) &&\text{(definition of ∪\cup∪)}\ &\IFF x\in ((A\cup B)\cap(A\cup C)) &&\text{(definition of ∩\cap∩)} \end{align} The second distributive law for sets follows in exactly the same way.

\medbreak

While ∪\cup∪ is analogous to \OR\OR\OR and ∩\cap∩ is analogous to \AND\AND\AND, we have not yet seen any operation is set theory that is analogous to the logical ``not'' operator,\NOT\NOT\NOT. Given a set AAA, it is tempting to try to define {x\st\NOT(x∈A)}\{x\st \NOT(x\in A)\}{x\st\NOT(x∈A)}, the set that contains everything that does not belong to~AAA. Unfortunately, the rules of set theory do not allow us to define such a set. The notation {x\stP(x)}\{x\st P(x)\}{x\stP(x)} can only be used when the domain of discourse of PPP is a set, so there must be an underlying set from which the elements that are/are not in AAA are chosen, i.e. some underlying set of which AAA is a subset. We can get around this problem by restricting the discussion to subsets of some fixed set. This set will be known as the \nw{universal set}. Keep in mind that the universal set is only universal for some particular discussion. It is simply some set that is large enough to contain all the sets under discussion as subsets. Given a universal set UUU and any subset AAA of UUU, we can define the set {x∈U\st\NOT(x∈A)}\{x\in U\st \NOT(x\in A)\}{x∈U\st\NOT(x∈A)}.

\begin{definition} Let UUU be a given universal set, and let AAA be any subset of UUU. We define the \nw{complement} of AAA in UUU to be the set A‾\overline{A}A that is defined by A‾={x∈U\stx∉A}\overline{A}=\{x\in U\st x\not\in A\}A={x∈U\stx​∈A}. \end{definition}

Usually, we will refer to the complement of AAA in UUU simply as the complement of AAA, but you should remember that whenever complements of sets are used, there must be some universal set in the background.

Given the complement operation on sets, we can look for analogs to the rules of logic that involve negation. For example, we know that p\AND\NOTp=Fp\AND\NOT p=\Fp\AND\NOTp=F for any proposition~ppp. It follows that for any subset AAA of UUU, \begin{align} A\cap\overline{A} &= {x\in U\st (x\in A)\AND (x\in \overline{A})} &&\text{(definition of ∩\cap∩)}\ &= {x\in U\st (x\in A)\AND (x\not\in A)} &&\text{(definition of complement)}\ &= {x\in U\st (x\in A)\AND \NOT(x\in A)} &&\text{(definition of ∉\not\in​∈)}\ &= \emptyset \end{align} the last equality following because the proposition (x∈A)\AND\NOT(x∈A)(x\in A)\AND \NOT(x\in A)(x∈A)\AND\NOT(x∈A) is false for any xxx. Similarly, we can show that A∪A‾=UA\cup\overline{A}=UA∪A=U and that A‾‾=A\overline{\overline{A}}=AA=A (where A‾‾\overline{\overline{A}}A is the complement of the complement of AAA, that is, the set obtained by taking the complement of~A‾\overline{A}A.)

The most important laws for working with complements of sets are DeMorgan's Laws\index{DeMorgan's Laws} for sets. These laws, which follow directly from DeMorgan's Laws for logic, state that for any subsets AAA and BBB of a universal set~UUU, \begin{align} \overline{A\cup B}&=\overline{A}\cap\overline{B}\ \intertext{and} \overline{A\cap B}&=\overline{A}\cup\overline{B} \end{align} For example, we can verify the first of these laws with the calculation \begin{align} \overline{A\cup B}&= {x\in U\st x \not\in (A\cup B)} &&\text{(definition of complement)}\ &= {x\in U\st \NOT( x\in A\cup B)} &&\text{(definition of ∉\not\in​∈)}\ &= {x\in U\st \NOT(x\in A \OR x\in B)} &&\text{(definition of ∪\cup∪)}\ &= {x\in U\st (\NOT(x\in A))\AND(\NOT(x\in B))} &&\text{(DeMorgan's Law for logic)}\ &= {x\in U\st (x\not\in A) \AND (x\not\in B)} &&\text{(definition of ∉\not\in​∈)}\ &= {x\in U\st (x\in\overline{A}) \AND (x\in\overline{B})} &&\text{(definition of complement)}\ &= \overline{A}\cap\overline{B} &&\text{(definition of ∩\cap∩)} \end{align}

An easy inductive proof can be used to verify generalized versions of DeMorgan's Laws for set theory.
(In this context, all sets are assumed to be subsets of some unnamed universal set.) A simple calculation verifies DeMorgan's Law for three sets: \begin{align} \overline{A\cup B\cup C}&=\overline{(A\cup B)\cup C}\ &=\overline{(A\cup B)}\cap\overline{C} &\text{(by DeMorgan's Law for two sets)}\ &=(\overline{A}\cap\overline{B})\cap\overline{C} &\text{(by DeMorgan's Law for two sets)}\ &=\overline{A}\cap\overline{B}\cap\overline{C} \end{align} From there, we can derive similar laws for four sets, five sets, and so on. However, just saying ``and so on'' is not a rigorous proof of this fact. Here is a rigorous inductive proof of a generalized DeMorgan's Law:

\begin{theorem} For any natural number n≥2n\geq 2n≥2 and for any sets X1X_1X1​, X2X_2X2​, \dots, XnX_nXn​, [\overline{X_1\cup X_2\cup \cdots \cup X_n} = \overline{X_1} \cap \overline{X_2} \cap\cdots\cap \overline{X_2}] \end{theorem} \begin{proof} We give a proof by induction. In the base case, n=2n=2n=2, the statement is that X1∪X2‾=X1‾∩Xn‾\overline{X_1\cup X_2}=\overline{X_1}\cap\overline{X_n}X1​∪X2​​=X1​​∩Xn​​. This is true since it is just an application of DeMorgan's law for two sets.

For the inductive case, suppose that the statement is true for n=kn=kn=k. We want to show that it is true for n=k+1n=k+1n=k+1. Let X1X_1X1​, X2X_2X2​, \dots, Xk+1X_{k+1}Xk+1​ be any kkk sets. Then we have: \begin{align} \overline{X1\cup X_2\cup \cdots \cup X{k+1}} &= \overline{(X1\cup X_2\cup \cdots \cup X_k) \cup X{k+1}}\ &= \overline{(X1\cup X_2\cup \cdots \cup X_k)}\cap\overline{X{k+1}}\ &= (\overline{X1}\cap\overline{X_2}\cap\cdots\cap\overline{X_k})\cap\overline{X{k+1}}\ &= \overline{X1}\cap\overline{X_2}\cap\cdots\cap\overline{X{k+1}} \end{align} In this computation, the second step follows by DeMorgan's Law for two sets, while the third step follows from the induction hypothesis. \end{proof}

Just as the laws of logic allow us to do algebra with logical formulas, the laws of set theory allow us to do algebra\index{algebra} with sets. Because of the close relationship between logic and set theory, their algebras are very similar. The algebra of sets, like the algebra of logic, is Boolean algebra.\index{Boolean algebra!in set theory} When George Boole wrote his 1854 book about logic, it was really as much about set theory as logic. In fact, Boole did not make a clear distinction between a predicate and the set of objects for which that predicate is true. His algebraic laws and formulas apply equally to both cases. More exactly, if we consider only subsets of some given universal set UUU, then there is a direct correspondence between the basic symbols and operations of propositional logic and certain symbols and operations in set theory, as shown in this table:

\begin{center} \begin{tabular}{|c|c|} \hline \strut \textbf{Logic}& \textbf{Set Theory}\cr \hline \strut T\TT& UUU\cr \strut F\FF& ∅\emptyset∅\cr \strut p\ANDqp\AND qp\ANDq& A∩BA\cap BA∩B\cr \strut p\ORqp\OR qp\ORq& A∪BA\cup BA∪B\cr \strut \NOTp\NOT p\NOTp& A‾\overline{A}A\cr \hline \end{tabular} \end{center}

\noindent Any valid logical formula or computation involving propositional variables and the symbols T\TT, F\FF, \AND\AND\AND, \OR\OR\OR, and~\NOT\NOT\NOT can be transformed into a valid formula or computation in set theory by replacing the propositions in the formula with subsets of UUU and replacing the logical symbols with UUU, ∅\emptyset∅, ∩\cap∩, ∪\cup∪, and the complement operator.

Just as in logic, the operations of set theory can be combined to form complex expressions such as (A∪C)∩(B∪C‾∪D)‾(A\cup C)\cap\overline{(B\cup \overline{C} \cup D)}(A∪C)∩(B∪C∪D)​. Parentheses can always be used in such expressions to specify the order in which the operations are to be performed. In the absence of parentheses, we need precedence\index{precedence rule} rules to determine the order of operation. The precedence rules for the Boolean algebra of sets are carried over directly from the Boolean algebra of propositions. When union and intersection are used together without parentheses, intersection has precedence over union. Furthermore, when several operators of the same type are used without parentheses, then they are evaluated in order from left to right. (Of course, since ∪\cup∪ and ∩\cap∩ are both associative operations, it really doesn't matter whether the order of evaluation is left-to-right or right-to-left.) For example, A∪B∩C∪DA\cup B\cap C \cup DA∪B∩C∪D is evaluated as (A∪((B∩C))∪D(A\cup((B\cap C))\cup D(A∪((B∩C))∪D. The complement operation is a special case. Since it is denoted by drawing a line over its operand, there is never any ambiguity about which part of a formula it applies to.

The laws of set theory can be used to simplify complex expressions involving sets. (As usual, of course, the meaning of ``simplification'' is partly in the eye of the beholder.) For example, for any sets XXX and~YYY, \begin{align} (X\cup Y)\cap(Y\cup X)&=(X\cup Y)\cap(X\cup Y) &&\text{(Commutative Law)}\ &=(X\cup Y) &&\text{(Idempotent Law)} \end{align} where in the second step, the Idempotent Law, which says that A∩A=AA\cap A=AA∩A=A, is applied with A=X∪YA=X\cup YA=X∪Y. For expressions that use the complement operation, it is usually considered to be simpler to apply the operation to an individual set, as in~A‾\overline{A}A, rather than to a formula, as in A∩B‾\overline{A\cap B}A∩B. DeMorgan's Laws can always be used to simplify an expression in which the complement operation is applied to a formula. For example, \begin{align} A\cap \overline{B\cup\overline{A}} &= A\cap (\overline{B}\cap\overline{\overline{A}}) && \text{(DeMorgan's Law)}\ &= A\cap (\overline{B}\cap A) && \text{(Double Complement)}\ &= A\cap (A\cap\overline{B}) && \text{(Commutative Law)}\ &= (A\cap A)\cap \overline{B}) && \text{(Associative Law)}\ &= A \cap \overline{B} && \text{(Idempotent Law)} \end{align}

\medbreak

As a final example of the relationship between set theory and logic, consider the set-theoretical expression A∩(A∪B)A\cap (A\cup B)A∩(A∪B) and the corresponding compound proposition p\AND(p\ORq)p\AND(p\OR q)p\AND(p\ORq). (These correspond since for any xxx, x∈A∩(A∪B)≡(x∈A)\AND((x∈A)\OR(x∈B))x\in A\cap(A\cup B) \equiv (x\in A)\AND ((x\in A)\OR (x\in B))x∈A∩(A∪B)≡(x∈A)\AND((x∈A)\OR(x∈B)).) You might find it intuitively clear that A∩(A∪B)=AA\cap(A\cup B)=AA∩(A∪B)=A. Formally, this follows from the fact that p\AND(p\ORq)≡pp\AND(p\OR q)\equiv pp\AND(p\ORq)≡p, which might be less intuitively clear and is surprising difficult to prove algebraically from the laws of logic. However, there is another way to check that a logical equivalence is valid: Make a truth table. Consider a truth table for p\AND(p\ORq)p\AND(p\OR q)p\AND(p\ORq): \begin{center} \begin{tabular}{|c|c||c|c|} \hline ppp& qqq& p\ORqp\OR qp\ORq& p\AND(p\ORq)p\AND (p\OR q)p\AND(p\ORq)\ \hline \strut false& false& false& false\ false& true& true& false \ true& false& true& true \ true& true& true& true \ \hline \end{tabular} \end{center} The fact that the first column and the last column of this table are identical shows that p\AND(p\ORq)≡pp\AND(p\OR q)\equiv pp\AND(p\ORq)≡p. Taking ppp to be the proposition x∈Ax\in Ax∈A and qqq to be the proposition x∈Bx\in Bx∈B, it follows that the sets AAA and A∩(A∪B)A\cap (A\cup B)A∩(A∪B) have the same members and therefore are equal.
